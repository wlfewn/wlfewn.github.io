spring boot集成spring session

spring boot项目集成spring session,官方已经给了很详细的demo,并且集成redis
戳这里
其他形式使用spring session

所以，重点就落在redis的使用上
1、redis官方没有提供windows平台的安装包，所以如何安装查看这里
2、如果发现redis无法运行,可能是这个原因
3、redis密码管理,需要在配置文件添加或修改
4、测试spring session，参考这里

微服务下的使用
1、微服务集成spring session。想法是把spring session做成一个中间插件，需要使用时就引用。
一般需要注意项目配置的扫描包路径。
demo地址 https://github.com/wlfewn/study-demo/tree/master/cloud-session-demo
2、微服务集成spring session与zuul网关。zuul网关不需要引入spring session,但要注意routes配置请求头(sensitiveHeaders)
如果遇到每次请求的session id不一致，参考这篇博客 http://blog.csdn.net/bianchengninhao/article/details/78899591
demo 地址 https://github.com/wlfewn/study-demo/tree/master/cloud-session-zuul-demo
